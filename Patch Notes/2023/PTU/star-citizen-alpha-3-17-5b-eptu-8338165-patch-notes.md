# Star Citizen Patch 3.17.5b
Alpha Patch 3.17.5b has been released and is now available to test on the Experimental PTU!   Patch should now show: VERSION 3.17.5-EPTU.8338165.

It is strongly recommended that players delete their USER and Shader folders for the Public client after patching, particularly if you start encountering any odd character graphical issues or crash on loading. The USER folder can be found (in default installations) at C:\Program Files\Roberts Space Industries\StarCitizen\EPTU. The Shader folders can be found here C:\Users\%username%\AppData\Local\Star Citizen.

Audience: Wave 1
Server Region: US/EU

Please monitor sc-testing-chat in spectrum for updates and testing instructions.
USER.cfg settings: r_displaySessionInfo = 1

The primary objective with 3.17.5 is to inject the updated Lunar New Year envelope (Year of the Rooster for 2953), alongside a few fun goodies we plan to give out. These were initially in the 3.18 version, which is obviously not on the live servers yet, therefor we're releasing a quick patch (3.17.5) to include those in the live build, in preparation for the celebration kicking off this week.

This is a quick EPTU publish to test a couple bug fixes in a public environment with you all and to verify the integrity of the build compared to the current LIVE environment.
- Fixed - Red Festival - Shopping / Selling / Economy - The Envelopes are not able to be sold at admin office
- Fixed - Interactables / Narrative - The Horse variant of the Lunar New Year envelopes is incorrectly named as 'Red Festival Envelope'
