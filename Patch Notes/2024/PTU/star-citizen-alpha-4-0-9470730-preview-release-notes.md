# Star Citizen Alpha Patch 4.0
Alpha Patch 4.0 has now been released onto the PREVIEW Environment! Select the Preview channel on the launcher dropdown to play this build. Launcher should now show: VERSION 4.0-LIVE.9470730.

> Star Citizen Alpha 4.0 brings a full wipe and reset to all accounts. Everyone will start fresh with their account hangars and wallets +20,000 aUEC.

This release of 4.0 onto the Preview Channel marks the start of new persistence data and will continue on this build as if it was the live environment. While this preview channel runs, the previous 3.24.3 LIVE environment will remain, allowing players to play on both versions of the game. After the Preview Channel period ends, progress in the 4.0 build will be saved as we move it onto the LIVE channel.


# Build Info
- Long Term Persistence: Disabled (Full Persistence Wipe)
- Server Meshing: 5:5:500 Player Shards
- Starting aUEC: 20,000
