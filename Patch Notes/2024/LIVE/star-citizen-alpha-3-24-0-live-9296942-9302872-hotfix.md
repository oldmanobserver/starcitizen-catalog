# Star Citizen Alpha Patch 3.24.0 Hotfix
Alpha Patch 3.24.0 has been released onto the LIVE servers, and is now available to test! Patch should now show: VERSION 3.24.0-LIVE.9296942.


# Technical
This hotfix will not affect non-LTP items in game so you will not lose consumables, rentals, refinery jobs, character/hangar customization, and home locations.


# Bug Fixes
- Fixed - PU - Disconnection Error - 30009 - Players are disconnected while loading into PU - reason="Remote Disconnect - Cannot send the current solar system for 'SendCurrentSolarSystem'
May Require a Character Repair to fix accounts already getting a 30009 error https://robertsspaceindustries.com/account/reset
- Fixed - PU - Locations - Instanced Hangars / ATC - If a player calls ATC too quickly after their gateway approval window expires to land, when the hangar doors close the ship can get stored and eject the player
- Fixed - PU - Missions / Cargo - Hauling missions - Mission cargo crates are not appearing in the kiosk at the pick-up location
- Fixed - PU - Locations / Usables - MultiLocation - After spawning, pressing <F> will trap the player in bed
- Fixed - PU - Openable Cargo - Stor*All containers of all sizes can be retrieved from inventory access kiosks
