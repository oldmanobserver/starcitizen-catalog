# Star Citizen Patch 3.22.1 Hotfix
Alpha Patch 3.22.1 has been released and is now available to test!   Patch should now show: VERSION 3.22.1-EPTU.9097523-9101073.

It is strongly recommended that players delete their USER folder for the Public client after patching, particularly if you start encountering any odd character graphical issues or crash on loading. The USER folder can be found (in default installations) at C:\Program Files\Roberts Space Industries\StarCitizen\EPTU. 


Audience: All Backers
Long Term Persistence: Enabled
Server Info: EPTU Channel, US Servers only
Pledge Copy: Enabled

**Testing Focus**

This build is an Overdrive event related hotfix that we are intending to put on LIVE later this week and will be on PTU to test compatibility to make sure we don't have to clear out database information such as medpens, ammo, and other short term items.
