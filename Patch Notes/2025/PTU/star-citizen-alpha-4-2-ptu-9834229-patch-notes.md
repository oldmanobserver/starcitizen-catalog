# Features & Gameplay
> **Gameplay**
- Fix for loot containers not opening and also prevented players from being able to manually close them so players always know if a location has been looted

**Storm Breaker and ASD Location Polish**
Location LOD, Lighting, and Art Polish. AI Navigation and Behavior Polish. Location Music Improvements.

**Valakkar Polish**
Removed adult explosion radius (was not intended). Reduced Apex Valakkar explosion radius from 20m to 10m. Adjusted Apex Valakkar Non-weakspot damage multiplier to 1%. Stopped it so when you mined the Valakkar pearls everyone would die. Animation and Attack Movement Polish. Removed ability to scan the Apex Valakkar from above the storm with a ship. 

> **Ships and Vehicles**
- Added new Scanning Mode UX Updates for more clean information
- Updated all 300 series loadout kits to use 1:1 gimbal size setup to match current game design as originals used size down gimbals
> **Core Tech**
- Further Server and AI Navigation Performance Optimizations
