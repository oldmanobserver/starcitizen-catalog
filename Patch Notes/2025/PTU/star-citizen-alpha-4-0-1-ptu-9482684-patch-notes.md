# Star Citizen Alpha Patch 4.0.1
Alpha Patch 4.0.1 has been released onto the PTU!  Patch should now show: VERSION 4.0.1-PTU.9482684.

Characters in this new environment have been built from Long Term Persistence(LTP) data.
Currently with LTP, certain player states such as personal hangar layout, custom characters, and in-game purchased consumables are not saved between releases which will cause some items to be lost in the next update.

Server Info: PTU Channel - US/EU Servers
Audience: All Backers
Long Term Persistence: Enabled 
Mesh Configuration: 5:5:500
Hangar Spawning: Enabled
Starting aUEC: 15,000,000
