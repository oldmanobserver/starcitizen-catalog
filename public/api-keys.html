How to get an API key

This site uses your own LLM provider API key to answer questions. You only need a key from one provider to get started — pick whichever one you already have an account with, or the cheapest option for the model you want to try. You can always add more later.

Before you start

  • You will need to create an account with the provider (free) and add a payment method or buy credits.
  • Most providers charge per-token. A typical chat message here costs a fraction of a cent.
  • Treat your API key like a password — anyone with it can spend on your account. We store it encrypted at rest and only decrypt it in-memory to forward your prompt.
  • If you ever leak a key, return to the provider's dashboard and revoke / rotate it immediately.

OpenAI (GPT-4o, GPT-4.1, etc.)

  1. Go to platform.openai.com and sign in or create an account.
  2. Add a payment method or prepaid credits at Billing → Overview. (A new API account is separate from a ChatGPT Plus subscription — Plus does not include API credits.)
  3. Open platform.openai.com/api-keys.
  4. Click Create new secret key, give it a name like "Star Citizen Catalog", and click Create.
  5. Copy the key (starts with sk-…). You will only see it once.
  6. Paste it into Settings → API Keys → OpenAI.

Anthropic (Claude)

  1. Go to console.anthropic.com and sign in or create an account.
  2. Add credits under Settings → Billing. (A Claude.ai subscription does not include API credits.)
  3. Open Settings → API Keys.
  4. Click Create Key, give it a name, and click Create Key.
  5. Copy the key (starts with sk-ant-…). You will only see it once.
  6. Paste it into Settings → API Keys → Anthropic.

Google (Gemini)

  1. Go to aistudio.google.com and sign in with a Google account.
  2. Open aistudio.google.com/app/apikey.
  3. Click Create API key. If prompted, create or pick a Google Cloud project — any project is fine.
  4. Copy the key (starts with AIza…).
  5. Paste it into Settings → API Keys → Google.

Google AI Studio offers a generous free tier for Gemini models, with paid usage available if you enable billing on the Cloud project.

xAI (Grok)

  1. Go to console.x.ai and sign in with your X / xAI account.
  2. Add credits or a payment method under Billing.
  3. Open the API Keys page.
  4. Click Create API Key, give it a name, and click Save.
  5. Copy the key (starts with xai-…). You will only see it once.
  6. Paste it into Settings → API Keys → xAI.

OpenRouter (one key, many models)

OpenRouter is a single account that gives you access to models from many providers (OpenAI, Anthropic, Google, Meta, Mistral, and more). Good choice if you want to experiment with several models without signing up for each one.

  1. Go to openrouter.ai and sign in (Google / GitHub / email).
  2. Add credits under Settings → Credits.
  3. Open Settings → Keys.
  4. Click Create Key, give it a name, and click Create.
  5. Copy the key (starts with sk-or-…).
  6. Paste it into Settings → API Keys → OpenRouter.

Troubleshooting

  • "Invalid API key" / 401 errors: Double-check you copied the full key with no extra spaces, and that you pasted it under the correct provider.
  • "Insufficient quota" / 429 errors: You need to add credits or a payment method in the provider's dashboard. A free signup is usually not enough.
  • Model not listed: The provider may have deprecated it or you may not have access. Pick another model from the dropdown.
  • Want to rotate a key? Revoke the old one in the provider's dashboard, generate a new one, and click Replace in Settings.
