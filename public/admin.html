Admin jobs

Each button kicks off a GitHub Actions workflow on the main branch. Jobs run on GitHub's runners, not on this server — feel free to close the page.

Sync catalog metadata

Fast (~30s). Refreshes the D1 catalog tables (video series, dates, patch versions) so questions like "latest Star Citizen Live" find the right document. Does NOT touch the vector index. Run after adding new transcripts or patch notes.

Rebuild vector index

Incremental (default): seconds to a few minutes — only re-embeds source files whose contents changed since the last run, and deletes vectors for files that were removed. Full rebuild: ~30 minutes — re-embeds every chunk. Use only after changing chunking logic or the embedding model.

Look for new YouTube videos

Scans the RSI channel for new videos in the current year, downloads transcripts, and commits them back to this repo. Slow (~150s between transcript downloads). Leave a year empty to use the current year.

Look for new patch notes

Scrapes the Spectrum patch-notes forum and commits any new entries. Provide an x-rsi-token from your browser's DevTools (Network tab on any spectrum page) to also fetch Evocati NDA posts.

Retrieval diagnostic

Type a query to see exactly what the retriever does: which intent was matched, which focus documents (if any), and what chunks Vectorize returned. No LLM is called.
