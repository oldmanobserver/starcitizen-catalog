# Star Citizen Alpha Patch 4.0.1 LIVE
Alpha Patch 4.0.1 has now been released onto the LIVE Environment! Launcher should now show: VERSION 4.0-LIVE.9499080.


Star Citizen Alpha 4.0.1 comes built from LTP data so economy and purchases saved to LTP will not be wiped. Things such as medpens, ammo, some types of cargo, refinery jobs, and other consumables are not saved to LTP and will be removed.

# Build Info
