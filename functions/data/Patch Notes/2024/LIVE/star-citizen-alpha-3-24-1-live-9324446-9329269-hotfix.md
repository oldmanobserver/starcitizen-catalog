# Star Citizen Alpha Patch 3.24.1 Hotfix
Alpha Hotfix Patch 3.24.1 has been released onto the LIVE environment, and is now available to test! Patch should now show: VERSION 3.24.1-LIVE.9324446.

# Technical
This hotfix will not affect non-LTP items in game so you will **not **lose consumables, rentals, refinery jobs, character/hangar customization, and home locations.

# Client Fixes - Immediate Fixes
- Fixed a client crash related to interacting with Storage Kiosks
- Fixed a client crash in AC Vanduul Swam
