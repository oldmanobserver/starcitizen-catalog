# Star Citizen Alpha Patch 3.24.0 Hotfix
Alpha Patch 3.24.0 has been released onto LIVE servers, and is now available to test! Patch should now show: VERSION 3.24.0-LIVE.9296942.


# Technical
This hotfix will not affect non-LTP items in game so you will not lose consumables, rentals, refinery jobs, character/hangar customization, and home locations.
We intend to leave this on the HOTFIX channel for a bit to test comparability before pushing out to everyone on the main LIVE channel


# NEW FIXES
- Fixed cargo boxes clipping into ground right after detaching from the cargo grid (STARC-122789)
- Disabled setting a respawn point to a ship that is in an instanced hangar as a partial fix for player characters getting broken when the spawn point is streamed out (STARC-124234)
- Fixed 2 Server Crashes
