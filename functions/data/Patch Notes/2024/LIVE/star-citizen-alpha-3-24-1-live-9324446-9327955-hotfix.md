# Star Citizen Alpha Patch 3.24.1 Hotfix
Alpha LIVE Patch 3.24.1 has been released onto the LIVE environment, and is now available to test! Patch should now show: VERSION 3.24.1-LIVE.9324446.

# Technical
This hotfix will not affect non-LTP items in game so you will not lose consumables, rentals, refinery jobs, character/hangar customization, and home locations.


# Client Fixes - Immediate Fixes
- Fixed - 32 SCU Crates can clip through the Freight Elevator of the hangar in Baijini point
- Fixed -  PU - Social - AR markers - Party markers are not visible to other party members
- Fixed -  ARGO ATLS - The crosshair can get in a state where it does not update its position
- Fixed - Freight Manager kiosk icons scale excessively large at 4k
- Fixed - Restricted area warning sometimes shows incorrectly for ATLS
- Fixed  - ATLS tractor beam - Vehicles cannot be moved up or down, only rotated
- Fixed - PU - Crash - Using a / Commands in MobiGlas will result in CTD
 
 
# Server Fixes - May take time to propagate to new shards
- Fixed issue with some mission markers not appearing
- The ATLS will now be correctly impounded, and/or receives appropriate crime stats for blocking hangar exits and spawning of ships/vehicles
- Fixed - Perosnal hangar - ASOP/ Ship spawn - If on a vehicle and interacting with the ASOP in a hangar you can bypass the ship elevator animation and spawn or store ships instantly
- Fixed - PU - Stanton - Personal Hangar - Entering a ship while its being stored will not remove the player
- Fixed 2 Server Crashes
- Fixed a Server Deadlock
