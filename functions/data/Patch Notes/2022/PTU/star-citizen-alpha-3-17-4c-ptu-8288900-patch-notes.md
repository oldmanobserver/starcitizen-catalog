# Star Citizen Patch 3.17.4c
Alpha Patch 3.17.4c has been released to the PTU, and is now available to test!   Patch should now show: VERSION 3.17.4-PTU.8288900.

It is strongly recommended that players delete their USER and Shader folders for the Public client after patching, particularly if you start encountering any odd character graphical issues or crash on loading. The USER folder can be found (in default installations) at C:\Program Files\Roberts Space Industries\StarCitizen\PTU. The Shader folders can be found here C:\Users\%username%\AppData\Local\Star Citizen.

Database Reset: No
Long Term Persistence: Enabled 
Pledge Copy: Disabled (Players will receive a limited ship list)
Starting aUEC: 10,000

***Please monitor sc-testing-chat in spectrum for updates and testing instruction*s.**

USER.cfg settings: r_displaySessionInfo = 1
