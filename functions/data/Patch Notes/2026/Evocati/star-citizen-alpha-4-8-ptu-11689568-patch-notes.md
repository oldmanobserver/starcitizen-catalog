# Features & Gameplay
> Gameplay
**New Event: Return of Xenothre**at
A new upcoming event featuring the Xenothreat, this will have a Progression tracker and will reward blueprints to the player when they reach milestones on the aforementioned tracker.

**Tactical Strike Groups Updates**
Increased power relays and cooling units CS signature to allow other missiles to home in on them. Lowered core distortion damage. Lighting, LOD, ambience audio, fog, module explosion, and set dressing polish.

**Ships & Vehicles**
Vehicle Uneditable / Editable itemport sweep: Full item port audit for all ships to make sure ship components are at the correct setup for editable or uneditable
