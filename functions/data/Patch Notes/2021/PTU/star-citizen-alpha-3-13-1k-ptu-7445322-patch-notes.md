# Star Citizen Patch 3.13.1k
Alpha Patch 3.13.1k has been released to the PTU, and is now available for all backers to test! Patch should now show: VERSION 3.13.1-PTU.7445322.

It is strongly recommended that players delete their USER folder for the Public client after patching, particularly if you start encountering any odd character graphical issues or crash on loading. The USER folder can be found (in default installations) at C:\Program Files\Roberts Space Industries\StarCitizen\PTU. 

Audience: All Backers
Database Reset: YES
Long Term Persistence: Enabled
Pledge Copy: Enabled
Starting aUEC: 15,000,000

***Please monitor sc-testing-chat in spectrum for updates and testing instruction*s.**

USER.cfg settings: r_displaySessionInfo = 1


**Testing Focus**
-     General Stability
This build should make a noticeable difference in client stability along with a server crash fix. We also wanted to move the PU selection to be the first selection on the list which should make it an easier experience for new players.
